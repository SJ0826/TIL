# CSS란?

**Cascading**: 계단식<br>
**Style**: 멋을 내다<br>
**Sheets**: (종이)한 장<br>
<br>
즉, 계단식으로 스타일을 정의하는 문서입니다.<br>
<br>
css는 웹문서를 꾸며주기 위해 사용하는 언어입니다.<br>
<br>
확장자는 *.css입니다.<br>
<br>
css문서는 html문서와 함께 작업을 수행합니다.

```
<link href="style.css" rel="stylesheet">
```

이와 같이 html문서에 css문서의 링크를 걸어주어 작업을 수행하게 됩니다.
